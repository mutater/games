1) Download and install the latest version of Python from www.python.org/downloads.
   (Make sure to check the box talking about "PATH" and to associate ".py" files with Python)

2) Open run by pressing win + r, type "cmd" (without quotes), and press enter.

3) Type "python -m pip install pywin32 playsound pillow"

4) You should be good to go! Run "magyka.py" to play.
